# ユビキタスプロジェクション
#### 詳細：

**機能**
* 線の描写<br>
    -右クリックを押してる状態のみ線を描写．
* 線の色を変更<br>
    -eキーで線の色を黒色に変更．<br>
    -wキーで線の色を白色に変更．<br>
    -gキーで線の色を緑色に変更．<br>
    -bキーで線の色を青色に変更．
* 線の太さを変更<br>
    -pキーで線の太さを太くする．<br>
    -mキーで線の太さを細くする．
* 描写画面のリセット<br>
    -rキーで描写画面をリセット．
* 描写画面の保存<br>
    -sキーで描写画面を"output.png"として保存．
* udp通信とプログラムの終了<br>
    -qキーでudp通信とプログラムを終了．

**実装**
* C++
* OpenCV

**各ファイルの説明**
* Rx.cpp：受信側のプログラム
* Tx.cpp：送信側のプログラム
